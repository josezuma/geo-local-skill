# FAQ

## What is geo-local-skill?
Agent skill for local business AI discovery — GBP, LocalBusiness schema, NAP.

## How do I install it?
```bash
git clone https://github.com/josezuma/geo-local-skill.git
cd geo-local-skill
python3 scripts/generate.py --help
```

## Do I need any external dependencies?
No. All scripts use Python stdlib only.
